Hangman Game Description

This is a simple yet engaging JavaScript Hangman game. The player is challenged to guess a 5-letter word by selecting individual letters from an on-screen keyboard. The player has 7 lives, which are depleted for each incorrect guess. If the player correctly guesses the word before running out of lives, they win the game. If not, the game reveals the correct word and the player loses.

Function Descriptions

  • randomWord(): Fetches a random 5-letter word from the Random Word API and converts it to uppercase.
  • displayKeyboard(): Creates the on-screen keyboard with buttons for each letter of the alphabet.
  • displayMaskedWord(): Masks the selected word by replacing all its letters with underscores.
  • handleKeyPress(e): Handles the click event for each keyboard button, updating the displayed word and lives counter based on the player's guess.
  • disableKeyboard(): Disables all keyboard buttons when the game is over.
  • resetGame(): Resets the game by fetching a new word, re-enabling keyboard buttons, and resetting the lives counter.
  • updateHangmanImage(): Updates the hangman image based on the number of wrong guesses.
